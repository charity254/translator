# Translator — English ↔ Swahili

A simple full stack Django app for translating words and phrases between English and Swahili.

🌍 **Live site:** https://translator-81mi.onrender.com

## Run Locally

```bash
git clone https://github.com/charity254/translator.git
cd translator
pip install -r requirements.txt
python manage.py runserver
```

Visit `http://127.0.0.1:8000`

## Supported Phrases

| English | Swahili |
|---|---|
| hello | habari |
| buy maize | nunua mahindi |
| good morning | habari ya asubuhi |
| are you going to work? | unaenda kazini? |
| animals | wanyama |
| fruits | matunda |
| school | shule |
| go to bed | enda kulala |
| what do you want to eat? | unataka kula nini? |
| I have finished studying | nimemaliza kusoma |

Unknown words are returned unchanged.
